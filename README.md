# Luke Woydziak Personal Site

Static personal website for `luke.woydziak.com`, designed for GitHub Pages.

## Quick start

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Set the source to the main branch root.
5. Ensure your DNS for `luke.woydziak.com` points to GitHub Pages.
